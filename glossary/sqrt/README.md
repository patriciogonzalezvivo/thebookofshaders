## Sqrt
Return the square root of the parameter

### Declaration
```glsl
float sqrt(float x)  
vec2 sqrt(vec2 x)  
vec3 sqrt(vec3 x)  
vec4 sqrt(vec4 x)
```

### Parameters
```x``` specify the value of which to take the square root.

### Description
```sqrt()``` returns the square root of ```x```.
