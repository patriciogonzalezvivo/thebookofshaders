## Exp
Return the natural exponentiation of the parameter

### Declaration
```glsl
float exp(float x)  
vec2 exp(vec2 x)  
vec3 exp(vec3 x)  
vec4 exp(vec4 x)
```

### Parameters
```x``` specify the value to exponentiate.

### Description
```exp()``` returns the natural exponentiation of ```x```.
