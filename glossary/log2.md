## Log2
Return the base 2 logarithm of the parameter

```glsl
float log2(float x)  
vec2 log2(vec2 x)  
vec3 log2(vec3 x)  
vec4 log2(vec4 x)
```

### Parameters
```x``` Specify the value of which to take the base 2 logarithm.

### Description
```log2()``` returns the base 2 logarithm of ```x```.
