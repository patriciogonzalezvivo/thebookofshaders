## Log
Return the natural logarithm of the parameter

### Declaration
```glsl
float log(float x)  
vec2 log(vec2 x)  
vec3 log(vec3 x)  
vec4 log(vec4 x)
```

### Parameters
```x``` s### Declarationpecify the value of which to take the natural logarithm.

### Description
```log()``` returns the natural logarithm of ```x```.
