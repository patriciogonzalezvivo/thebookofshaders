## Abs
Return the absolute value of the parameter

```glsl
float abs(float x)  
vec2 abs(vec2 x)  
vec3 abs(vec3 x)  
vec4 abs(vec4 x)
```

### Parameters
```x``` Specify the value of which to return the absolute.

### Description
```abs()``` returns the absolute value of ```x```.
