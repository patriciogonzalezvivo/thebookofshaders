## Asin
Return the arcsine of the parameter

### Declaration
```glsl
float asin(float x)  
vec2 asin(vec2 x)  
vec3 asin(vec3 x)  
vec4 asin(vec4 x)
```

### Parameters
```x``` specify the value whose arcsine to return.

### Description
```asin()``` returns the angle whose trigonometric sine is ```x```.
